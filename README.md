# 📰 NewsApp 

A modern React-based News Application that fetches real-time news using NewsAPI and displays them with category filtering, infinite scrolling, and a clean responsive UI.

---

## 🚀 Features

- 🌍 Real-time news updates using NewsAPI
- 📂 Category-based filtering (Business, Sports, Technology, etc.)
- 🔄 Infinite scrolling for seamless browsing
- 📊 Top loading progress bar for better UX
- 🖼️ Responsive news cards with images, author, and date
- 🔗 Direct link to full news articles
- ⚡ Fast UI using React Hooks
- 🧭 Navigation using React Router DOM

---

## 🛠️ Tech Stack

- React.js (Functional Components & Hooks)
- React Router DOM
- Bootstrap 5
- NewsAPI
- React Infinite Scroll Component
- React Top Loading Bar

## ⚙️ Installation & Setup

### Clone the repository
git clone https://github.com/your-username/newsapp-react.git
cd newsapp-react

### Install dependencies
npm install

### Create .env file
REACT_APP_NEWS_API=your_newsapi_key_here

### Run the project
npm start
